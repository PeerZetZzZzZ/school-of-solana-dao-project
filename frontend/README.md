# Solana DAO frontend
### Prerequisites
node.js 20.10.0

### Installation
Optionally: use [NVM](https://github.com/nvm-sh/nvm) for Node.js version management.
```
cd dapp-scaffold
nvm install 20.10.0
nvm use 20.10.0
npm install
npm run dev
```
Application will be available on http://localhost:3000


### Production build
`npm run build`
